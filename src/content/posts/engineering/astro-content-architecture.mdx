这篇文章用于验证代码块、目录、行内代码与数学公式，不描述生产博客的真实实现细节。

## 稳定标识

内容模型可以把稳定标识与可读路由分开：

```ts
interface SamplePost {
  id: string
  slug: string
  featured: boolean
}
```

## 一个公式

布局需要容纳行内公式 $O(n)$，也要正确处理独立公式：

$$
T(n)=T(n-1)+1
$$

## 同步边界

主题仓库负责视觉、组件和交互；内容仓库负责文章、个人素材、站点身份与生产部署。样式提交验证后再同步到内容仓库。
